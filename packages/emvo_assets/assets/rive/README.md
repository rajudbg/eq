# Rive Mascot Files

Place .riv files here exported from Rive.app

Required files:

- mascot.riv: Main mascot with state machine (idle, listen, think, happy, concern, celebrate)
- mascot_idle.riv: Lightweight version for background/loading
- loading.riv: Loading animation

State Machine Inputs:

- idle (trigger)
- listen (trigger)
- think (trigger)
- happy (trigger)
- concern (trigger)
- celebrate (trigger)
